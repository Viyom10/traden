TITLE: TRADEN-PROD — Atomic Fee Enforcement in Decentralized Perpetual Trading on Solana

CONTEXT FOR THE GENERATOR:
You are designing a 10-slide final-evaluation deck for the BITS Pilani course
"BITS F452 — Blockchain Technology". The project is by Viyom Gupta
(2023A7PS0413G).

The project is a Solana-based decentralized perpetual futures trading platform
built on top of Drift Protocol. Its core academic contribution is a technique
called "atomic fee enforcement": the platform's 5-bps trading fee is bundled
into the same Solana transaction as the user's trade, so the user's single
Ed25519 signature (over a SHA-256 hash of the message) cryptographically binds
the two — they either both succeed or both revert. This is achieved entirely in
the application layer, with no new on-chain smart contract.
